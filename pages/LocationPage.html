Veja aqui mais informações sobre o desenvolvimento deste site:

Utilizei a tag “div” para criar todas as divisões entre áreas das páginas. Utilizei a tag "style" para adicionar os estilos de CSS aos elementos. Utilizei a tag “a” para adicionar hiperligações para redirecionar para outras páginas, como é visível na navbar que tem 5 páginas para onde podemos navegar. Utilizei a tag “h3” e “h4” para criar headers e dar destaque (sendo que o h4 é de importância que o h3), como é possível ver nos títulos de todas as páginas como por exemplo na Home Page onde está escrito “Sobre a Car Today.” Utilizei a tag “p” para criar parágrafos como é possível ver na Home Page na área “Sobre A Car Today”, sendo que utilizei a ferramenta Lorem para gerar o texto. Utilizei a tag “span” para adicionar conteúdo em linha. Utilizei a tag “svg” em conjunto com a tag “line” para criar uma linha divisória no fim de todas as páginas para poder colocar precisamente esta informação que está a ler agora. Utilizei a tag “img” para adicionar imagens às páginas como é possível verificar na página viaturas.
